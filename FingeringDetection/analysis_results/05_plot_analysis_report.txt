2. 박스플롯 (Box Plot)
--------------------------------------------------
🎯 목적: 중앙값, 사분위수, 이상치 비교

📈 해석 방법:
• 상자의 위치 → 중앙값(median) 비교
• 상자의 크기 → IQR(사분위범위), 변동성 비교
• 수염의 길이 → 데이터 범위, 극값 비교
• 점들(◦) → 이상치(outliers) 존재

4. Q-Q Plot (Quantile-Quantile Plot)
--------------------------------------------------
🎯 목적: 데이터가 정규분포를 따르는지 확인

📈 해석 방법:
• 점들이 직선에 가까우면 → 정규분포에 가까움
• S자 곡선 → 꼬리가 두꺼운 분포 (heavy-tailed)
• 역S자 곡선 → 꼬리가 얇은 분포 (light-tailed)
• 아래로 볼록 → 오른쪽 치우침 (right-skewed)
• 위로 볼록 → 왼쪽 치우침 (left-skewed)

--------------------------------------------------
🎯 목적: 누적확률로 분포 차이를 명확히 비교

📈 해석 방법:
• 곡선이 왼쪽에 있으면 → 더 작은 값들이 많음
• 곡선이 오른쪽에 있으면 → 더 큰 값들이 많음
• 곡선 간 최대 거리 → 분포 차이의 크기
• 가파른 상승 → 특정 구간에 데이터 집중

7. 통계 요약 해석
--------------------------------------------------
🎯 목적: 수치적 지표로 분포 특성 정량화

📈 주요 통계 지표 해석:

• 평균 (Mean): 전체적인 중심 경향
• 표준편차 (Std): 변동성의 크기
• 변동계수 (CV): 상대적 변동성 (CV = Std/Mean)
• 왜도 (Skewness): 분포의 비대칭성
  - 0에 가까우면 대칭적
  - 양수면 오른쪽 치우침 (긴 오른쪽 꼬리)
  - 음수면 왼쪽 치우침 (긴 왼쪽 꼬리)
• 첨도 (Kurtosis): 분포의 뾰족함
  - 0에 가까우면 정규분포와 유사
  - 양수면 뾰족한 분포 (두꺼운 꼬리)
  - 음수면 평평한 분포 (얇은 꼬리)

🔍 현재 데이터 분석:

Left Hand 비교:
  평균 차이: +0.0232 (+2.2%)
  변동계수: Playing 0.095 vs Quiet 0.296
  → Quiet이 더 변동적 (일관성 낮음)
  ⭐ 실용적으로 작은 차이 (효과크기 작음)

Right Hand 비교:
  평균 차이: +0.0099 (+0.9%)
  변동계수: Playing 0.192 vs Quiet 0.206
  → 유사한 변동성
  ⭐ 실용적으로 무시할 수 있는 차이

8. 정규성 검정 해석
--------------------------------------------------
🎯 목적: 통계적 가설검정 방법 선택의 기준

📈 Shapiro-Wilk Test 해석:
• H0 (귀무가설): 데이터가 정규분포를 따른다
• H1 (대립가설): 데이터가 정규분포를 따르지 않는다
• p-value > 0.05 → 정규분포 가정 (모수검정 가능)
• p-value ≤ 0.05 → 비정규분포 (비모수검정 권장)
